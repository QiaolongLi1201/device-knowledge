# Host Software Knowledge

Data-first knowledge module for general desktop and frontend engineering.

The package is intentionally host-software focused and family neutral. It covers
Electron, React, Vite, TypeScript, and Node.js backend documentation anchors,
plus prompt fragments and failure hints for common desktop/frontend workflows.
It does not include hardware-specific content.

```ts
import { hostSoftwareKnowledgeModuleData } from '@device-knowledge/host-software-knowledge';
```
